# Sopa
**A Stopwatch implement in C kinda like [sowon](https://github.com/tsoding/sowon)**

## Dependencies
- [SDL2](https://www.libsdl.org/)

- [SDL_image.h](https://www.libsdl.org/projects/SDL_image/)

### Quick Start
## Linux
```bash
./build.sh
./sopa `flags` `time`
```
### Examples
```bash
./sopa 69420
```
```bash
./sopa -r 2001s29h05m
```
```bash
./sopa -rp 69:420:96
```
