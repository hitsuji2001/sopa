# Sopa
**A Stopwatch implement in C kinda like [sowon](https://github.com/tsoding/sowon)**

## Dependencies
[SDL2](https://www.libsdl.org/)

### Quick Start
## Linux
```bash
$ ./build.sh
$ ./sopa `flags` `time`
```
### Examples
```bash
$ ./sopa 69420
$ ./sopa -r 2001s29h05m
$ ./sopa -rp 69:420:96
```
